# IPL Data Analysis Dashboard

## Overview
Interactive Power BI dashboard analyzing 636 IPL matches 
across 10 seasons (2008-2017)

## Key Insights
- Mumbai Indians are the most successful team
- CH Gayle leads Player of Match awards
- M Chinnaswamy Stadium hosted most matches

## Tools Used
- Power BI
- Python
- Excel
